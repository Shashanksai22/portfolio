Event Voting Machine

Revolutionizing the landscape of election systems, my project centers on the creation of an Event Voting Machine. Employing Verilog and Vivado, I meticulously designed a sophisticated system utilizing a Zed board equipped with five buttons. Four of these buttons represent individual participants, allowing voters to cast their preferences, while the fifth button serves a dual purpose for both resetting the system and calculating the final votes.

Under the hood, the Verilog code implemented in Vivado governs the logic of the voting machine. The code intricately manages the functionalities of the buttons, ensuring a seamless and secure voting process. The innovative design accommodates up to four participants, each associated with a dedicated button, complemented by LEDs for transparent vote displays. This project not only underscores my proficiency in hardware description languages and FPGA programming but also showcases my commitment to enhancing the reliability and security of essential democratic processes.
